    Text Animator #

    api

    添加两个组件 TextAnimator TextAnimatorPlayer

    1.常用标签 #

    动画演示

    "fade"
    <fade>我们把研究对象统称为元素</fade>
    显示之后消失

"speed"
    <speed=0.02>字体显示速度<speed=1>have fun bring
    字体显示速度, speed 越小越慢

"shake"
    <shake>震动</shake>

"wiggle"
    <wiggle>晃动</wiggle>

"bounce"
    <bounce>Oh, how lucky</bounce>

"expl"
    <expl>分裂到处跳在回来</expl>

"rot"
    <rot>旋转</rot>

"swing"
    <swing>左右晃</swing>

"share"
    <share>像浪一样</share>

"rainb"
    <rainb>渲染彩色字</rainb>

"incr"
    <incr><rainb>渲染颜色字体晃动</rainb></incr>

    2.富文本 # 

    "颜色"
    <color=red>Behaviors</color>
    <#00ff00>Green text</color>
    <mark=#ffff0040>阴影强调</mark>

"基础"
    <b>粗体</b>
    <i>斜体</i>
    <u>下划线</u>
    <s>删除线</s>

    <sup>上标</sup>
    <sub>下标</sub>
    <size=48>设置字体大小</size>
    <size=+18>相对字体大小</size>
    <size=-18>相对字体大小</size>

"偏移距离"
    <pos=5.5>Chara
