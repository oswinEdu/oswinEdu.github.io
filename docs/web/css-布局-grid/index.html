    CSS-布局-grid #

    参考

    参考

    1.属性 #

    • justify-items: 水平位置 start end center

    • align-items: 垂直位置 start end center

    • place-items: 垂直 水平

    • grid-template-columns: 列

        1.可分很多列, 设置每列宽度
  2.如分成3列, 5个子元素, 多两个第二行显示
  3.minmax(150px, 25%) 1fr

  grid-template-columns: minmax(150px, 25%) 1fr;
      第一列的宽度是minmax(150px, 25%) 最小宽度为150px，最大宽度为总宽度的25%
      第二列为1fr，即所有剩余宽度

    • grid-template-rows: 行

        grid-template-rows: auto 1fr auto;
      第一部分 和第三部分 的高度都为auto，本来的内容高度；第二部分的高度为1fr，剩余的所有高度
  1.可分成很多行, 设置每行高度

    • grid-template: 行 列

        grid-template: auto 1fr auto / auto 1fr auto;

    • grid-row-gap:

        设置行间距

    • grid-column-gap:

        设置列间距

    2.属性修饰 #

    • auto-fill: .container { display: grid; grid-template-columns: repeat(auto-fill, 100px); } 每列宽度 100, 行放不下自动换下一行, 行宽不固定

    • fr

        grid-template-columns: 1fr 1fr; 第一列占 1/2 第二列占 1/2
  grid-template-columns: 2fr 1fr; 第一列占 2/3 第二列占 1/3
  grid-template-columns: 150px 1fr 2fr; 150固定 剩下1/3 剩下2/3
