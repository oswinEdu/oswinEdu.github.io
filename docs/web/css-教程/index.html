    3.css中级 #

    1.布局-display #

    • 删除: display:none
    • 隐藏: visibility:hidden
    • block: 一个元素占一行
    • inline: 可多个元素占一个行, 不允许在元素上设置宽高
    • inline-block: 允许在元素上设置宽高

    2.布局-position #

    • static: 不受 top、right、bottom 和 left 约束
    • relative: 相对定位, 受 top、right、bottom 和 left 约束
    • fixed: 固定定位, 页面滚动, 它不动
    • absolute: 绝对定位
    • sticky: 滚动完悬浮顶端
    • z-index: 层级
