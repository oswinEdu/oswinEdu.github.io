    Hugo #

    中文站点

    主题

    1.安装 #

    参考

    1.hugo命令 #

    参考

        brew install hugo
    hugo version

    hugo new site blog      创建站点
    cd blog                 文件夹
    hugo server             启动本地服务器
    hugo server -D
    hugo server -p [端口] 
    hugo                    生成静态 web 文件上传github访问

    hugo new about.md       创建Markdown文件, 模板 xxx/archetypes/default.md
    hugo new post/first.md

    2.站点文件夹 #

    • archetypes：存放用 hugo 命令新建的 md 文件应用的 front matter 模版
    • content：存放内容页面，如 Blog
    • layouts：存放定义网站的样式，写在layouts文件下的样式会覆盖安装的主题中的 layouts文件同名的样式
    • static：存放所有静态文件，如图片
    • data：存放创建站点时 Hugo 使用的其他数据
    • public：存放 Hugo 生成的静态网页
    • themes：存放主题文件
    • config.toml：网站配置文件

    3.github显示博客 #

    • hugo new site blog 创建本地博客
    • cd blog 执行 hugo 命令生成发布文件在 public文件夹
    • blog文件夹一个github地址忽略public文件夹
    • public文件夹真正静态网页上传 github 对应 https://oswinedu.github.io

    2.安装主题hugo-book #

    hugo-book

    1.配置主题 # 

        1. 下载 hugo-book主题
    2. 把 hugo-book 文件夹拷贝到 xxx/themes
    3. 拷贝 hugo-book/exampleSite/content.en 到 xxx/content (menu 文件夹可删除 2022年废弃)
    4. 拷贝 /hugo-book/exampleSite/assets 到 xxx/assets
    5. 拷贝 /hugo-book/exampleSite/resources 到 xxx/resources
    6. 拷贝 hugo-book/static 到 xxx/static
    7. 配置 config.toml 主题 theme = 'hugo-book' 
    8. hugo server -D 启动本地服务
    • content.en 文件夹内容
    • config.toml
        baseURL = '<http://example.org/>'
    languageCode = 'en-us'
    title = 'My New Hugo Site'
    theme = 'hugo-book' 设置主题
    • 启动本地服务器后

    2.左侧side #

    • 点击站点显示 xxx/content/_index.md 内容
    • xxx/content 每个文件夹下都应该有对应 _index.md 点击文件夹显示
    • 自动遍历 xxx/content 下所有文件夹和文件, 显示左侧侧边栏

    3.config.toml #

    配置可参考 hugo-book/exampleSite/config.toml

        baseURL = 'http://example.org/'
    languageCode = 'en-us'
    title = 'My New Hugo Site'
    # 配置主题
    theme = 'hugo-book' 

    # 开启katex渲染
    [markup.goldmark.renderer]
      unsafe = true

    4.xxx.md配置 # 

    ---
title: aaaa123                      左侧边栏显示名字，不配做默认 xxx.md 文件名
type: docs
bookCollapseSection: true           文件夹下 _index.md 默认文件夹不展开如 Collapsed 文件夹
bookFlatSection: true               文件夹下 _index.md 文件夹不可点击如 Shortcodes 文件夹

draft: true                         true执行hugo命令静态文件不会被导出到public文件夹
weight: 1                           多个文件显示顺序,小的在前面
---
