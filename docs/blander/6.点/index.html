    点 #

    1.基础操作 #

    1.移动 #

    选中点 + g

    2.旋转 #

    选中点 + r

    3.缩放 #

    选中点 + s

    4.滑移 #

    选中点 + gg 或 shift + v
    滑移点沿着某条边移动

    5.删除 #

    选中点 + x 点所在的边和面也会被删除

    6.融并 #

    选中点 + ctrl + x

    2.连接顶点 #

    1.填充 #

    选中多个点 + f 填充

    2.连接-f #

    选中两个未连接点 + f
    仅仅是连接两个点，不会切割点所在的面

    3.连接-j #

    选中两个未连接点 + j 切割两个点所在的面

    4.连接-k #

    切割任意地方，生成点和面

    3.合并顶点 #

    相邻或重叠的顶点合并成一个

    1.合并-多个顶点 #

    选中多个顶点 + 右键/顶点合并

    2.合并-自动 #

    两个顶点距离小于设定值自动合并

    3.选中重叠顶点 #

    开启透视预览

    4.顶点倒角 #

    shift + ctrl + b

    1.参数-宽度 #

    2.参数-段数 #

    3.参数-形状 #

    4.倒角-两边 #

    5.倒角-三边 #

    6.倒角-四边 #

    5.顶点挤出 #

    1.挤出 #

    选中一个点 + e

    2.扩展 #

    选中一个点 + alt + d
    复制出一个新的顶点可以移动
