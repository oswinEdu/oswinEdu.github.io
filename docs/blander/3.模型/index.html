    模型 #

    1. 由 点、线、面 组成(主要是面)
    2. 面 由三边形、四边形、多边形组成，一般使用四边形面
    3. 模型内部中空

    1.模型基础 #

    1.认识模型 #

    平面、立方体、椎体、柱体、环体、球体

    2.添加模型 #

    3.微调模型 #

    首次创建模型左下角会出现, 微调面板, 调整模型

    4.设置模型属性 #

    (1)调整模型-侧边栏(快捷键n)

    1. 位置
    2. 旋转
    3. 缩放
    4. 尺寸

    (2)物体属性-右下角

    2.编辑模型 #

    1. 选中模型按下 TAB 键进入编辑模式，再按 TAB 键退出编辑模式
    2. 编辑模式对模型点、线、面操作

    1.单选点-线-面 #

    选择点|线|面模式，单击鼠标左键

    2.多选点-线-面 #

    1. 增加：按住 shift 键，鼠标左键单击未选择
    2. 减少：按住 ctrl 键，鼠标左键单击已选择
    3. 全选：选择点|线|面 按 a 键

    3.工具选点-线-面 #

    调整、框选、刷选、套索

    左上角工具栏

    3.选中-点|线|面 #

    0.选择 #

    1.并排边 #

    ctrl+alt+鼠标左键

    2.并排面 #

    ctrl+alt+鼠标左键

    3.循环边 #

    选中一条边，在选择里找循环边

    4.间隔边|面 #

    5.选中相连项 #

    ctrl + l
