    基础操作 #

    1.移动旋转缩放 #

    1.移动 #

    点击移动图标

    1. 沿着 x y z 轴移动
    2. 沿着 xy xz yz 三个平面移动
    3. 自由移动

    快捷键移动-g(grab抓取)

    1. 可以任意移动
    2. x y z 沿对应轴移动

    2.旋转 #

    点击旋转图标

    1. 可选择 x y z 轴旋转

    快捷键旋转-r(rotate)

    1. 可以任意旋转
    2. x y z 沿对应轴旋转

    3.缩放 #

    点击缩放图标

    1. 沿着 x y z 轴缩放
    2. 沿着 xy xz yz 三个平面缩放

    快捷键旋转-s(scale)

    1. 可以任意缩放
    2. x y z 沿对应轴缩放

    4.移动-旋转-缩放-归零 #

    1. alt + g 还原移动
    2. alt + r 还原旋转
    3. alt + s 还原缩放

    2.复制删除抓取 #

    1.复制 #

    复制cmd + c 黏贴cmd + v

    2.删除 #

    1. 右键删除
    2. delete 键直接删除
    3. x 确认删除

    3.复制抓取 #

    shift + d 再按 x y z 对应轴移动
