    工作区 #

    1.常用操作 #

    平移：cmd + 鼠标左键(后改的快捷键)

    旋转：option + 鼠标左键

    缩放：滑动鼠标滚轮

    2.坐标 #

    右手坐标系，z轴向上、xy平面右上角正，unity y轴向上、xz平面右上角正

    3.视图 #

    1.透视图 #

    近大远小、滚动鼠标滑轮查看近大远小

    2.正交视图 #

    无论远近一样大小，分别从6个正方向预览模型

    点击 Z轴 从上面预览模型、ctrl+Z轴 下面预览模型

    点击 X轴 从左面预览模型、ctrl+X轴 右面预览模型

    点击 Y轴 从前面预览模型、ctrl+Y轴 后面预览模型

    3.局部视图 #

    当有多个模型，只编辑一个或几个模型时，选中模型按 / 进入局部视图，退出再按 /

    4.预览 #
